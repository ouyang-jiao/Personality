# 🐉 奶龙人格测试

一款基于 Vue 3 的趣味人格测试网站，以可爱的奶龙形象为主题，包含 13 道测试题和 13 种人格类型。

## ✨ 功能特点

- 🎮 **13 道趣味测试题** - 全部围绕奶龙的故事设定
- 📊 **13 种人格类型** - 每种都有官方伪装版和真实整蛊版
- 🥚 **惊喜彩蛋** - 测试完成后可解锁彩蛋
- 📱 **响应式设计** - 支持手机和电脑访问
- 🎨 **奶龙风格** - 淡黄色主题，可爱的奶龙形象

## 🚀 在线预览

🌐 https://chimerical-youtiao-c68954.netlify.app/

## 🛠️ 技术栈

- **前端框架**: Vue 3 + JavaScript
- **UI 组件库**: Ant Design Vue 4.1
- **构建工具**: Vite 5.2
- **路由管理**: Vue Router 4
- **样式**: CSS3 + SCSS

## 📦 安装与运行

```bash
# 克隆仓库
git clone https://github.com/ouyang-jiao/Personality.git

# 进入项目目录
cd Personality

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
